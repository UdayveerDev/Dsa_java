#  DSA in Java

Welcome to my Data Structures and Algorithms workspace! This repository contains my Java implementations of core data structures, algorithms, and solutions to coding problems. 

##  About This Repository
This repo serves as my personal tracker for software engineering interview preparation and consistent coding practice. It includes optimal solutions to problems categorized by specific algorithmic patterns, complete with logic explanations and time/space complexity analysis.

## 📂 Repository Structure (Pattern-Wise)
The code is organized logically by algorithmic patterns and data structures to build strong problem-solving intuition:

* **Array Patterns:**
  * Two Pointers
  * Sliding Window
  * Kadane's Algorithm
  * Prefix Sum
  * Merge Intervals
* **Searching & Hashing:**
  * Binary Search (1D, 2D, Answer on Search Space)
  * Hashmaps
* **Core Data Structures:**
  * Linked List (Slow/Fast Pointers, Reversal, Cycle Detection)
  * Stacks (Next Greater Element)
* **Advanced Data Structures:**
  * Heaps / Priority Queues (Top K elements, Merge K sorted)
  * Trees & Binary Search Trees (Traversals, LCA, Diameter, Path Sum)
  * Graphs (BFS, DFS, Topological Sort, Dijkstra, Bellman-Ford, Prim's)
* **Algorithmic Paradigms:**
  * Recursion & Backtracking
  * Dynamic Programming (1D DP, 0/1 Knapsack, Tabulation/Memoization)
